1. Ensure MongoDB and MongoDB Compass are installed
2. Ensure mongod is running in a terminal
2. Create a new connection in MongoDB Compass with the following connection string: mongodb://localhost:27017
3. Open the connection and click on create database
4. Name it TheeStooges
5. Go to the index tab
6. Create a new index
7. Call it username
8. Set the type to text
9. Expand options and tick Create Unique Index
10. use node server.js in the root folder to run the server.
